# Machine Learning Model for Smart Maintenance of Rotary Kiln Incinerator

## Description
Comparative Evaluation of ML Algorithms for Predictive Maintenance of Rotary Kiln Incinerator

## Contents
- Code for model training and evaluation
- Synthetic industrial dataset
- Performance results and visualizations

## Models Used
- Artificial Neural Network (ANN)
- Support Vector Machine (SVM)
- Decision Tree
- Naive Bayes

## Features
- Temperature
- Pressure
- Vibration
- Acoustic signals
- Flue gas emissions (CO, NOx, SOx, HCl, PM)

## Status Classes
- Normal
- Warning
- Failure

## Usage
Run scripts in `/code` folder to reproduce results.
